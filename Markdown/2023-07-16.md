## **10-3 크기 설정하기**

### **똑같은 크기로 배치하기 - IntrinsicWidth, IntrinsicHeigth** 
```dart
    Container(
    color: Colors.yellow,
    child: IntrinsicWidth(
    child: Column(
    crossAxisAlignment: CrossAxisAlignment.stretch,
    children: [
            Container(
            color: Colors.red,
            width: 230,
            height: 50,
            ),
            Container(
            color: Colors.green,
            width: 200,
            height: 200,
            ),
            Container(
            color: Colors.blue,
            width: 100,
            height: 100,
            ),
        ],
    ),
),
),
```
> **crossAxisAlignment의 값을 CrossAxisAlignment.stretch로 설정까지 해야한다.**

### 최소, 최대 범위로 배치하기 - ConstrainedBox

- **마지노선 정하기**
```dart
ConstrainedBox(
          constraints: const BoxConstraints(
              minWidth: 200, maxHeight: 300, minHeight: 30), //최소 넓이: 200, 최대 높이: 300 최소 높이: 30
          child: Container(
            color: Colors.green,
            height: 30,
            width: 150,
          ),

```

## **10-4 비율로 배치하기 - Expended**

**Expanded 위젯을 사용하면 width의 크기와 관계없이 같은 비율로 나오게 된다.**
```dart
  Row(
  children: <Widget>[
    Expanded(
        child: Container(
      color: Colors.red,
      width: 50,
    )),
    Expanded(
        child: Container(
      color: Colors.green,
      width: 150,
    )),
    Expanded(
        child: Container(
      color: Colors.yellow,
      width: 350,
    ))
  ],
),
//width가 각각 50, 150, 350이지만 Expanded를 사용했기 떄문에 1:1:1비율로 나오게 된다.
```
---
**등분 비율 설정하기 예시**
> **flex를 사용하면 위젯의 각각의 비율을 설정 가능하다.**
```dart
 Row(
    children: <Widget>[
      Expanded(
          flex: 2,
          child: Container(
            color: Colors.red,
            width: 50,
          )),
      Expanded(
          flex: 1,
          child: Container(
            color: Colors.green,
            width: 550,
          )),
      Expanded(
        flex: 2,
        child: Container(
          color: Colors.yellow,
          width: 50,
        ),
      ),
    ],
  ),
```
**만약 Expanded와 Expanded가 아닌 위젯을 함께 추가하면 어떻게 될까?**
> Expanded가 아닌 위젯은 설정된 크기대로 유지되며 그 다음 나머지 Expanded위젯이 비율에 따라 나누어 차지한다.
