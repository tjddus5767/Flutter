#### **체크박스 - Checkbox**
> **체크박스는 사용자에게 true나 false를 입력받는 위젯이다.<br> value 속성으로 체크박스의 값을 설정, onChanged 속성으로 상태 변경 시 호출할 이벤트 콜백**

```dart
Row(
    children: [
    Checkbox(
        value: isChecked,
        onChanged: (bool? value) {
            setState(() {
            isChecked = value;
            });
        }),
    Text(
        'Checkbox value is $isChecked',
        style: const TextStyle(fontSize: 30),
    ),
```
